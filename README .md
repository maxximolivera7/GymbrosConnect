# GymBrosConnect

Plataforma digital interactiva diseñada para la comunidad de fitness y powerbuilding. El objetivo principal de este sitio web es conectar a entusiastas del entrenamiento (Gymbros / Spotters) en zonas de Buenos Aires como Caballito y Villa Crespo, permitiéndoles coordinar entrenamientos seguros, compartir sus marcas personales en los tres grandes levantamientos (Sentadilla, Press de Banca y Peso Muerto) y descubrir los mejores gimnasios de la zona.

## Estructura del Proyecto

- `index.html`: Página de inicio y punto de entrada principal del sitio.
- `/css`: Contiene los estilos de diseño general y diseño responsive (`style.css`).
- `/pages`: Subpáginas del sitio (`academia.html`, `mapa.html`, `perfiles.html`, `rutinas.html`).
- `/imagenes`: Archivos de imagen y recursos visuales optimizados para la web.
- `/js`: Lógica e interacciones del sitio web.

## Tecnologías Utilizadas

- HTML5 (Estructura semántica)
- CSS3 (Estética brutalista, Grid, Flexbox y Media Queries para Responsive)
- JavaScript (Validación e interacción de formularios en tiempo real)
